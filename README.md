# 🏠 AI Listing Generator

**MLS-ready property descriptions in seconds — powered by Azure OpenAI GPT-4o**

A Streamlit web app that generates polished, professional real estate listing
descriptions from structured property data. Built for Saskatchewan real estate
agents who want consistent, high-quality copy without the writing time.

## ✨ Features

- **Instant Descriptions** — Enter property details, get a polished MLS-ready
  listing in under 30 seconds
- **Multiple Variations** — Generate up to 3 unique versions per listing and
  pick the best one
- **Smart Property-Type Prompts** — Specialized writing guidance for 7 property
  types: Bungalow, Two-Storey, Bi-Level, Condo/Apartment, Townhouse, Acreage,
  Mobile Home
- **Tone Selection** — Professional, Family-Friendly, Luxury, Investor-Focused,
  or First-Time Buyer
- **Saskatchewan Focus** — Trained for local neighbourhoods, property styles,
  and market language
- **No Clichés** — Prompt engineering eliminates "hidden gem", "must-see",
  "dream home", and 13 other overused phrases
- **Download & Copy** — One-click download as text or copy to clipboard
- **Canadian Data Residency** — Hosted on Azure Canada East — data never leaves
  the country
